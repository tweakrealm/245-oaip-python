# Основы алгоритмизации и программирования на Python
Репозиторий содержит мои лабораторные работы
## Структура репозитория
- `Lab1/` — лабораторная работа №1
- `Lab2/` — лабораторная работа №2
- `Lab3/` — лабораторная работа №3
- и так далее…

Каждая папка содержит:
- исходный код
- файл с отчетом (`.pdf` или `.docx`)
- инструкции по запуску (если требуется)

## Как запускать
1. Перейти в папку с нужной лабораторной
2. Открыть проект в IDE или скомпилировать через командную строку
3. Следовать инструкциям в файле `README` внутри каждой лабораторной (если есть)

## Примечания
- Все работы выполнялись самостоятельно
- Код комментирован для удобства проверки
